Hide Product Price, Add to Cart Button and Product Qty on Shop Odoo Apps

Hide-Disable Price, Qty and Add to Cart Option on Shop Odoo App helps users to disable product details on the website. Users can configure or tick on disable product details on shop/product page based on that select disable option for non login/public user and all users. Users can disable product price, qty option and add to cart button on shop/product page.

Features

Configure Disable Product Details on Shop/Product page

Under website configuration user can configure disable product details on shop/product page.

Disable Product Details for Non Login and All Users

User can select disable option for public/non login user and all users to disable add to cart button, qty option and product price.

Configuration for Disable Product Details on Shop/Product page

Goto Website -> Configuration -> Settings in that tick on 'Disable details on shop/product page' checkbox.

When you tick on 'Disable details on shop/product page' checkbox, disable options are visible for 'public/non login user' and 'All users'

You can tick on any of the disable option like 'Disable Add to cart button', 'Disable Product price', 'Disable Qty Option' checkbox.

Disable Product Details for Non Login User

You can see based on cofiguration add to cart button and qty option are disable only for public/non login user.

You can see for login user, add to cart button, price and qty option are visible.

You can set disable product details for all users from website configuration.

Disable Product Details for All Users

You can see based on configuration add to cart button, qty option and price are disable for public/non login and also login user.
